# Compressed Tokens Overview

Welcome to the Compressed Tokens section. This page provides an overview of compressed tokens in ZK Compression.

## What are Compressed Tokens?

Compressed tokens are a key component of ZK Compression that enable efficient token operations while maintaining security and verifiability.

## Key Features

- **Efficiency**: Reduced on-chain storage requirements
- **Cost-effective**: Lower transaction fees
- **Scalability**: Handle more transactions with less resources
- **Compatibility**: Works with existing Solana token standards
