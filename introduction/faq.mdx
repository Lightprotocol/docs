## Frequently Asked Questions

### General Questions

<AccordionGroup>
  <Accordion title="What is ZK Compression?">
    ZK Compression is a technology that allows for efficient compression of data on the Solana blockchain using zero-knowledge proofs, significantly reducing storage costs while maintaining security and verifiability.
  </Accordion>

{" "}
<Accordion title="How does ZK Compression reduce costs?">
  By compressing account data and using zero-knowledge proofs to verify state
  transitions, ZK Compression can reduce storage costs by up to 99% compared to
  traditional on-chain storage.
</Accordion>

  <Accordion title="Is ZK Compression compatible with existing Solana programs?">
    ZK Compression is designed to be compatible with existing Solana infrastructure while providing additional compression capabilities through specialized SDKs and tools.
  </Accordion>
</AccordionGroup>

### Technical Questions

<AccordionGroup>
  <Accordion title="What are compressed tokens?">
    Compressed tokens are SPL tokens that use ZK Compression to store their state efficiently while maintaining full functionality and composability with other Solana programs.
  </Accordion>

{" "}
<Accordion title="How do I create compressed tokens?">
  You can create compressed tokens using the ZK Compression SDK. See our [Token
  Metadata guide](/integrate/token-metadata) for detailed instructions.
</Accordion>

  <Accordion title="What programming languages are supported?">
    Currently, we provide SDKs for TypeScript/JavaScript and Rust. Other language bindings may be available in the future.
  </Accordion>
</AccordionGroup>
