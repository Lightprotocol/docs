## getCompressionSignaturesForTokenOwner

Returns the signatures of the transactions that have modified an owner's compressed token accounts. This is a paginated endpoint

### Body

<ParamField body="id" type="string" required>
  An ID to identify the request. **Possible values:** `test-account`
</ParamField>

<ParamField body="jsonrpc" type="string" required>
  The version of the JSON-RPC protocol. **Possible values:** `2.0`
</ParamField>

<ParamField body="method" type="string" required>
  The name of the method to invoke. **Possible values:**
  `getCompressionSignaturesForTokenOwner`
</ParamField>

<ParamField body="params" type="object" required>
  Request parameters for compression signatures by token owner
</ParamField>
