## getCompressedAccount

Returns the compressed account with the given address or hash

### Body

<ParamField body="id" type="string" required>
  An ID to identify the request. **Possible values:** `test-account`
</ParamField>

<ParamField body="jsonrpc" type="string" required>
  The version of the JSON-RPC protocol. **Possible values:** `2.0`
</ParamField>

<ParamField body="method" type="string" required>
  The name of the method to invoke. **Possible values:** `getCompressedAccount`
</ParamField>

<ParamField body="params" type="object" required>
  Request for compressed account data
  
  **Default:** `{"address":null,"hash":"11111111111111111111111111111111"}`
  
  **Example:** `{"address":null,"hash":"11111111111111111111111111111111"}`
</ParamField>
