## getTransactionWithCompressionInfo

Returns the transaction data for the transaction with the given signature along with parsed compression info

### Body

<ParamField body="id" type="string" required>
  An ID to identify the request. **Possible values:** `test-account`
</ParamField>

<ParamField body="jsonrpc" type="string" required>
  The version of the JSON-RPC protocol. **Possible values:** `2.0`
</ParamField>

<ParamField body="method" type="string" required>
  The name of the method to invoke. **Possible values:**
  `getTransactionWithCompressionInfo`
</ParamField>

<ParamField body="params" type="object" required>
  Request parameters
</ParamField>

### Response

<ResponseField name="200" type="object">
  A Solana transaction with additional compression information
  <Expandable title="Response">
    <ResponseField name="compression_info" type="object">
      Compression information for the transaction
    </ResponseField>
    <ResponseField name="transaction" type="object">
      An encoded confirmed transaction with status meta
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="429" type="object">
  Exceeded rate limit.
</ResponseField>

<ResponseField name="500" type="object">
  The server encountered an unexpected condition that prevented it from
  fulfilling the request.
</ResponseField>
