## getMultipleCompressedAccounts

Get multiple compressed accounts in a single request.

### Parameters

- `accountHashes` (array): Array of account hashes

### Example Request

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getMultipleCompressedAccounts",
  "params": [["hash1", "hash2", "hash3"]]
}
```

### Example Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "accounts": [
      {
        "hash": "hash1",
        "data": "base64EncodedData1",
        "owner": "ownerPublicKey1",
        "lamports": 1000000,
        "executable": false
      },
      {
        "hash": "hash2",
        "data": "base64EncodedData2",
        "owner": "ownerPublicKey2",
        "lamports": 2000000,
        "executable": false
      }
    ]
  }
}
```
